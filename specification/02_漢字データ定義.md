# 漢字データ定義

## データクラス定義

```kotlin
data class KanjiItem(
    val id: Int,
    val kanji: String,
    val reading: String,
    val sentence: String,
    val category: KanjiCategory
)

enum class KanjiCategory {
    NUMBERS,    // 数字
    DAYS,       // 曜日
    DIRECTIONS, // 方向・位置
    NATURE,     // 自然・風景
    PEOPLE,     // 人・大きさ
    BODY,       // 体・動作
    COLORS,     // 色・形
    THINGS      // もの・その他
}
```

## 選択肢生成ロジック

選択肢は消去法で解けないよう、同じカテゴリから優先的に選ぶ:

```kotlin
fun generateChoices(correctKanji: KanjiItem): List<String> {
    val choices = mutableListOf(correctKanji.reading)
    
    // 同じカテゴリから選択肢を取得
    val sameCategory = KanjiData.getByCategory(correctKanji.category)
        .filter { it.reading != correctKanji.reading }
        .map { it.reading }
        .shuffled()
    
    for (reading in sameCategory) {
        if (choices.size >= 4) break
        if (reading !in choices) choices.add(reading)
    }
    
    // 同カテゴリで足りない場合は他カテゴリから補充
    if (choices.size < 4) {
        val otherReadings = KanjiData.allKanji
            .filter { it.category != correctKanji.category }
            .map { it.reading }
            .shuffled()
        
        for (reading in otherReadings) {
            if (choices.size >= 4) break
            if (reading !in choices) choices.add(reading)
        }
    }
    
    return choices.shuffled()
}
```
